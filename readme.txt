Требования к проекту:
-Python3
-git


Для запуска проекта необходимо выполнить следующие действия.

1.Клонировать или скачать этот репозиторий
2.Вызвать консоль и перейти в папку проекта
	
	cd <path_to_repo>/Market

3.Запустить установку необходимых пакетов для Python из файла requirements.txt.

	pip install -r requirements.txt

4.Создать суперпользователя в базе.

	python manage.py createsuperuser

6.Запускаем проект
	
	python manage.py runserver

В проекте есть данные для теста. Они хранятся в базе.
Никто не мешает добавить доп данные в админке.
